FizzBuzz: A Coding Challenge

Given three values, to determine the number to count to. When a number is divisible by the first value, display Fizz, when the number is divisible by the second value, display Buzz. And finally display Fizz-Buzz when the number is divisible by both values.

Instruction

Enter a first fizz value, a buzz value and the total number you want to display. Press the FizzBuzz button, and the munbers in the stop range of values will change to Fizz, Buzz, and FizzBuzz. Warning! Even when the value of the stop value can reach up to 5000, try not to put too large a number in this field in order to avoid server slowness.
